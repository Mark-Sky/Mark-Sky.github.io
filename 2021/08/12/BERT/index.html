前言

BERT这个东西涉及的相关概念真的是好多，先大概列一下

  • Transformer
  • self-attention
  • pre-trained model
  • fine-tune
  • end-to-end
  • NSP （Next Sentence Prediction）
  • Masked LM（Masked Language Model）

概念

Downstream tasks

downstream tasks is what the field calls those supervised-learning tasks that utilize a pre-trained model or component

下游任务就是那些利用预训练的模型的监督学习任务

ULM-FiT

Universal Language Model Fine-tuning，标志着模型迁移在NLP领域成为可能。先对模型在普遍的数据集上面训练参数，然后把模型转移到特定的任务上面时，对该任务的数据集进行参数微调。而在自然语言处理的微调需要一些奇淫技巧，像是论文里面列举的

  • discriminative fine-tuning
  • Slanted triangular learning rates
  • Gradual unfreezing

Transformer

Transformer是一种编码-译码的神经网络模型，该模型对于长依赖关系的处理比LSTM要更好

具体看Transformer的相关笔记

OpenAI Transformer

该模型只用到了Transformer的译码部分

Pre-training a Transformer Decoder for Language Modeling

OpenAI Transformer只是一个前向语言模型，也就是说它不能对单词的左右的上下文都作为辅助信息来判断

bi-directional

与之对应的是Forward language model和Backward Language model
